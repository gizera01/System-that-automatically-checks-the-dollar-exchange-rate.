# Dollar Exchange Rate Bot with Selenium

A simple web automation project developed in Python to fetch and check the real-time commercial USD to BRL exchange rate.

## 🎯 Project Goal
This script was built to automate the daily task of checking currency exchange rates. To bypass common anti-bot triggers (like CAPTCHAs) on traditional search engines, the bot accesses structured public data from AwesomeAPI, extracts the latest price, and prints the formatted result on the screen.

## 🛠️ Tech Stack
* **Python 3**
* **Selenium** (Web navigation and element scraping)
* **JSON** (Handling the fetched structured data)

## 📦 Installation and Usage

1. Make sure you have Python installed on your machine.
2. Install the Selenium package using your terminal:
   ```bash
   pip install selenium
   ```
3. Run the Python script:
   ```bash
   python your_script_name.py
   ```

## 📊 Expected Terminal Output
Once the script finishes running, you will see this output in your terminal:
```text
[Bot de Cotação Executado com Sucesso]
Valor comercial atual do Dólar: R\$ 5.42
```
